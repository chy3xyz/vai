# VAI Examples 使用说明

本目录包含若干可以直接运行的示例，用于快速了解 VAI 的能力和典型用法。

> 运行所有示例前，请先在仓库根目录完成编译：
>
> ```bash
> cd /path/to/vai          # 仓库根目录
> v -prod .
> ```

## 1. Chat Bot（本地交互聊天）

文件：`examples/chat_bot.v`

功能：在终端中启动一个简单的聊天机器人，会调用你配置的 LLM（如 OpenRouter / OpenAI / Ollama）。

运行方式：

```bash
cd /path/to/vai
v run examples/chat_bot.v
```

建议环境变量（任选一种 LLM 配置好即可）：

```bash
export OPENROUTER_API_KEY="your_openrouter_key"
export VAI_DEFAULT_MODEL="gpt-4o-mini"
# 或使用本地 Ollama：确保 ollama serve 已经在运行
```

## 4. Telegram Bot 示例

文件：`examples/telegram_bot.v`

功能：说明如何基于 VAI 创建一个简单的 Telegram 机器人，接收消息并调用 LLM 回复。

运行方式（需提前在 BotFather 中创建机器人并拿到 token）：

```bash
cd /path/to/vai
export TELEGRAM_BOT_TOKEN="your_telegram_bot_token"
v run examples/telegram_bot.v
```

## 通用排错提示

- 如果示例运行时报「找不到模型 / 无法连接 LLM」：
  - 检查是否正确设置了 `OPENROUTER_API_KEY` / `OPENAI_API_KEY` 等环境变量。
- 如果网络请求超时：
  - 检查本机网络或代理设置，确保可以访问对应的 API 域名。
- 如果想查看更详细日志：
  - 可以在运行命令前导出 `RUST_LOG` / `VAI_LOG_LEVEL` 之类的变量（视你自己后续扩展而定）。
